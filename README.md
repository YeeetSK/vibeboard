# VibeBoard

Desktop kanban for Cursor Agent work across projects.

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](./LICENSE)
[![Platform](https://img.shields.io/badge/platform-macOS%20%7C%20Windows-lightgrey)](https://github.com/YeeetSK/vibeboard/releases)

![VibeBoard main board view](./docs/media/vibeboard-preview.gif)

## Features

- Project tabs → local folders, custom lanes, drag-and-drop cards
- Statuses: idle, running, needs attention, done
- Task detail: chat + syntax-highlighted diffs
- Agent CLIs: Cursor, Claude Code, Codex (pick in sidebar)
- Run modes: shared / branch / worktree
- Auto-move finished tasks through Review → Done
- Global search (projects, tabs, tasks, prompts)
- Desktop notifications + optional ntfy
- Fully local and private
- **Mac:** notch overlay (running / done / inactive, expand to reply)
- **Mac:** keyboard backlight flash alerts
- Auto-update from GitHub Releases (packaged builds)

## Download

[Releases](https://github.com/YeeetSK/vibeboard/releases): macOS `.dmg`, Windows `.exe`

Needs [Cursor](https://cursor.com) + `agent` CLI (`agent login` if prompted).
